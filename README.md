# TinyPic Lite | 极速图片压缩助手

Project extracted from `portfolio` and refactored as a standalone Vue 3 + Vite project.

## Features

- **Lightweight**: Pure Vue 3 + TailwindCSS. No Nuxt dependency.
- **Privacy First**: All processing happens locally in the browser using WebAssembly/JS.
- **Support**: JPG, PNG (with UPNG.js), WebP, AVIF.
- **Theme**: Dark/Light mode toggle.

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- Vue 3
- Vite
- Tailwind CSS
- Heroicons
